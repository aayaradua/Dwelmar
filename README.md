# Dwelmar

**A real estate marketplace connecting agents and buyers across Northern Nigeria, with map-based property search, location intelligence, and real-time chat.**

---

## Features

- Interactive map showing property listings
- Filter properties by state, LGA, or neighborhood
- Display environmental and risk layers (flood zones, land use)
- Heatmaps showing property price density
- Agent dashboard to submit and manage listings
- Admin approval and location verification
- Property details with images, price, and description
- Real-time chat between buyers and agents
- Smart search by area or drawn polygon

---

## Tech Stack

- **Frontend:** React, React-Leaflet / Mapbox
- **Backend:** Node.js, Express.js, Socket.IO
- **Database:** MongoDB (GeoJSON support)
- **GIS:** QGIS (for map layers and spatial data)
- **Storage:** Cloudinary or AWS S3 (for property images)
- **Map Tiles / Layers:** OpenStreetMap, GeoJSON
